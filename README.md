# Social Media API

## Overview
This project is a backend API for a social media platform built using Django and Django REST Framework.  
It includes user authentication, registration, login, token generation, and custom user model.

## Features
- Custom User Model (bio, profile picture, followers)
- Token Authentication
- User Registration & Login Endpoints
